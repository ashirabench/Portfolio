# What Explains the Gender Care Gap?

Do higher state unionization rates affect the male-female gap in unpaid care minutes? Does it differ by household structure?

We merged individual-level American Time Use Survey (ATUS) data with state-level union membership rates, built a gender care gap measure, and tested whether unionization moderates the gap. It doesn't — but family structure and education do.

**Course:** EC 422 . Michigan State University
**Authors:** Ashira Benchimol, Amiera Saanan, Lillian Davidson

## Key Findings

- Women spend about 73 more minutes per day on unpaid care than men (coef = 72.99, p < 0.001)
- Having a child under 5 dramatically widens the gap
- Union membership and coverage rates do not significantly moderate the gender care gap (controlling for state and year fixed effects)
- Education level and family structure (spouse present, children) are the strongest predictors

## A Note on Housewives

One of the most interesting things we found: housewives spend nearly double the care time of other women (~290 vs ~165 min/day). 
They pull the female average up significantly. This raises a question worth thinking about — is the "gender care gap" really a gender gap, or is it partly a labor force participation gap? 
If you removed housewives from the sample, the 73-minute gap would shrink. 
The gap is still real, but the size of it depends on who you include and why. We think that's worth questioning.

## Data

- **ATUS (American Time Use Survey)** — individual-level time use data (2003-2024) from [IPUMS](https://www.atusdata.org/)
- **Union Membership by State** — state-level union membership and coverage rates (1983-2024) from [unionstats.com](https://www.unionstats.com/)

Raw data files are too large for GitHub. See `RawData/` for source info.

## How to Run

Set your working directory to the project root in `README.do`, then:

```stata
do "README.do"
```

This runs all scripts in order: data build, descriptives, regressions.

## Methods

1. **Data Build** — Cleaned ATUS microdata (dropped unnecessary variables, created gender/marriage/children dummies, computed total unpaid care time). Cleaned union data (converted rates from strings to numeric, created state FIPS codes, collapsed to state-year level). Merged both datasets.

2. **Descriptive Analysis** — Care time distributions, gender trends over time, care gap by state, housewives vs. other women, care gap by education level, care vs. family income (polynomial fit), and an exploratory industry-level breakdown.

3. **Regressions** — Progressive OLS specifications: gender only, then adding marriage status, children, education, and finally union membership/coverage with state and year fixed effects. All regressions use survey weights.
